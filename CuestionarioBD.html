Preguntas y Justificaciones

  1. ¿Qué es una base de Datos - DB?

    2. Se puede definir Base de Datos como el conjunto de datos archivados que comparten vínculos entre sí, y se administran de manera que se evita la repetición o pérdida de los mismos.

  2. ¿En qué casos es conveniente usar bases de datos?

    3. En todo momento en el que se utilizarán datos que se puedan consultar, editar o eliminar por distintos usuarios o aplicaciones

  3. ¿Qué es un sistema de gestión de base de datos - DBMS?

    4. Como su nombre lo dice, es una base de datos acompañada a su vez de un conjunto de programas para acceder de manera específica a los recursos deseados, permitiendo así una administración eficiente de la colección.

  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.
    • Crear y organizar la base de datos.
    • Establecer las rutas de acceso, para que los datos puedan ser rápidamente accesados.
    • Manejar los datos, según lo solicite el usuario.
    • Registrar la actividad y el uso de la base de datos.
    • Respaldo y Recuperación, pues en caso de haber alguna falla en el sistema, se debe de poder recuperar los datos con facilidad.
    • Control de concurrencia, pues la concurrencia de usuarios no debe causar inconsistencia en los datos.
    • Seguridad e Integirdad. Una DBMS -necesita asegurarse que los datos no se vean perjudicados por cambios no autorizados.
  5. Ejemplifica y justifica en qué proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

    6. En el trabajo, al momento de elaborar las licecnias de aptitudes laborales DC3, se trabaja más facilmente cuando cuentas con una base de datos real y no con una base de datos de escritorio, pues llegan ocasiones en las que la información no concuerda y se duplica.
